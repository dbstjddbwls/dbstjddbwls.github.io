React Mini Project

  • 주제 : React로 뉴욕타임즈 API를 활용한 검색 서비스 개발
  • 구현 목표
    • Input에 키워드를 입력
    • 입력한 키워드는 최근 검색목록에 저장. 최신순으로 최대 5개
    • 최근 검색 목록은 input에 값을 입력할 때 보여져야함
    • Input 입력 후 0.5초동안 추가 입력이 없으면 검색 진행. Focus가 되면 보여지는 기능
    • 검색되면 화면에 해당 키워드가 포함된 뉴스 리스트를 띄운다.
    • 리스트에는 타이틀 헤더 내용들과 Clip 버튼, See Detail 버튼이 있어야한다.
    • Clip 버튼을 누르면 unClip 버튼으로 바뀌고, 해당 리스트는 /Clip/ 경로로 이동 시 보여져야함.
    • See Detail 버튼을 누르면 상세 페이지로 이동
    • unClip 버튼을 누르면 Clip 페이지에서 사라짐
    • infinite scroll 기능. 스크롤이 마지막에 닿았을 때 다음 Page 요청
    • 첫 페이지가 화면의 높이를 모두 못 채워도 추가로 요청해서 채워야함. 즉 다음 페이지를 불러오면 됨
      (API 호출 시 페이지 하나분의 기사가 전달됨. 즉 추가 요청)
