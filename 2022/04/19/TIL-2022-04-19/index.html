수업 내용

CSS 연결 방법

  • HTML에 link Tasg를 통하여 css 파일을 연결
  • CSS 파일 내부에 @import 를 통하여 css파일끼리 연결

선택자

    • : 모든 요소를 선택
  • . : HTML 내 class 속성의 요소를 선택

  • : HTML 내 id 속성의 요소를 선택

  • Selector1.Selector2 : 선택자 Selector1 과 Selector2 을 동시에 만족하는 요소를 선택
  • Selector1 > Selector : 앞 선택자 내 바로 밑 자식 요소를 선택
  • Selector Selector : 앞 선택자 내 모든 자식 요소를 선택
  • Selector + Selector : 앞 선택자 형제 요소를 선택

가상 클래스 선택자

  • :hover : 해당 요소에 마우스를 올리고 있는 동안
  • :active : 해당 요소를 클릭하고 있는 동안
  • :focus : 해당 요소에 포커스 가능한 요소에 포커스되고 있는 동안
  • :first-child : 형제 요소 중 첫 번째 요소를 선택
  • :last-child : 형제 요소 중 마지막 요소를 선택
  • :nth-child(n) : 형제 요소 중 n번째 요소를 선택
  • :not(Selector) : Selector 요소에 해당되지 않는 요소를 선택

가상 요소 선택자

  • ::before : 해당 요소의 내부 앞에 내용을 삽입
  • ::after : 해당 요소의 내부 뒤에 내용을 삽임

선택자 속성

  • [ABC] : 속성 ABC를 포함한 요소 선택
    ex) [disabled], [type]
  • [ABC=”DEF”] : 속성 ABC를 포함하는 요소 중 DEF 속성을 가진 요소 선택
    ex) [type=”text”], [data-attr-attr=”attr”]

회고

금일은 CSS 온라인 수업과 같이 JS특강도 진행하였으나 해당 특강의 경우 JS의 기본 문법이나 개념에 대해 설명하는 부분이였습니다. 특강의 경우 이미 알고있는 내용들이 주류여서 다시 한 번 복습한다는 생각으로 신청하였고 따로 노션에 기재하지는 않았습니다.

CSS 온라인 강의의 경우 아직 초반부라 간단한 용어와 선택자, 이용법에 대해 다뤘으며 이에 대해서도 아는 내용이지만 이 경우 시간이 지나 헷갈리거나 할 수 있는 부분을 위주로 정리하였습니다.
