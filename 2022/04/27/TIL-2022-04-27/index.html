JS 데이터의 기능들

  • String.match() : 정규표현식을 인자로 받으며, 해당 표현식에 맞는 문자 앞까지 추출한다.
  • String.trim() : 해당 문자열 안에 포함된 공백 문자를 제거하여 반환한다.
  • Number.toFixed() : 해당 숫자값에 소수점이 있다면 인자로 받는 소숫점 자리에서 끊어서 문자형으로 반환한다.
  • parseInt() : 문자열을 받으면 숫자형만 추출하여 반환한다. 단 이 과정에서 소숫점은 사라지고 정수만 반환한다.
  • parseFloat() : parseInt와 같으나 소숫점 자리까지 보여준다.
  • Math() : 수학적인 상수와 함수를 위한 내장 객체이다.
  • Math.abs() : 주어진 숫자의 절대값을 반환
  • Math.ceil() : 주어진 숫자값을 올림처리를 함
  • Math.floor() : 주어진 숫자값을 내림처리를 함
  • Math.round() : 주어진 숫자값을 반올림처리를 함
  • Math.random() : 0~1 사이의 랜덤한 난수를 추출한다.
  • Array.find() : 주어진 판별 함수나 조건에 의해 매칭되는 첫 요소를 반환한다.

  • Array.forEach() : 주어진 배열의 item 갯수만큼 특정한 콜백 함수를 반복 실행한다.

  • Array.map() : 주어진 배열의 item 갯수만큼 특정한 콜백 함수를 반복 실행하여 해당 값을 배열로 반환한다.

  • Array.filter() : 주어진 배열의 item 갯수만큼 특정한 콜백 함수를 반복 실행하여 해당 함수 조건에 맞는 item을 배열로 새롭게 만들어 반환한다.

  • Array.findIndex() : 주어진 배열의 item 갯수만큼 특정한 콜백 함수를 반복 실행하여 해당 배열 내 조건에 맞는 item의 첫 번째 요소의 index number를 찾는다.

  • Array.includes() : 주어진 배열 내 인자로 받은 데이터가 존재하는지 확인한다.

  • Object.assign() : 인자로 받은 객체 데이터를 합쳐서 비교 또는 서술할 때 사용할 수 있다.

구조 분해 할당

어떠한 객체의 구조를 분해햐여 변수로 할당한다. 정의되어있지 않은 속성을 할당하면 undefined를 반환하게 된다. 또한 따로 값을 지정하여 추가할 수 있고, 만약 기존에 값이 있는 경우 변수에 할당한 값은 무효처리 된다.
