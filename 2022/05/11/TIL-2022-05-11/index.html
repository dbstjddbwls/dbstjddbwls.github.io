boiler-plate

변경 없이 계속하여 재사용을 할 수 있는 저작물을 말한다. 개발 환경에서 사용되면 CRA, Next.js나 Gatsby 등이 존재한다.

CRA로 react 시작하기

  1. 터미널 실행
  2. npx create-react-app {folder-name} —template typescript
    (타입 스크립트로 진행)
  3. 설치 확인

TypeScript를 컴파일하기

CRA로 TypeScript, .tsx를 사용하면 굳이 기존 ts를 사용했던 것 처럼 npm를 별도로 받아 컴파일할 필요가 없다. CRA도 보일러 플레이트이기 때문에 자동으로 tsc 변환과 적용을 해준다.

회고

오늘은 토이프로젝트 발표를 끝내고 저녁 식사 이후 실시간 리액트 강의를 진행하였습니다. 이전 tsc를 조금 더 파보고 싶다는 생각이 있었는데 마침 강사님께서 React 수업은 tsc를 사용하여 작업한다 하셔서 매우 만족하였습니다. 다만 tsc의 경우 온라인 수업으로 한 두번 들은게 끝이라 이후 별도로 tsc에 대해 복습이 필요할 것 같습니다.
