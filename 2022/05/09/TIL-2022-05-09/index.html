토이 프로젝트(2)

이번 토이 프로젝트는 최대한 메인 홈 화면의 js 구조 작업에 집중하였습니다. 다만 코드를 보았을 때 너무 중복되는 코드가 많습니다. 하나의 function으로 만들고 인자를 2~3개를 받아 구분하여 작업하였으면 훨씬 좋지 않을까란 생각이 듭니다. 지금 다시 코드를 짠다면 특정 구간을 date, month, year로 구분하여 인자를 하나 주고, 두번 째 인자로는 해당 날짜의 값을 넣어 사용한다면 좀 더 간결하게 줄일 수 있을 것 같습니다.

항상 함수의 재활용성을 간과하고 코드를 짜는 것 같습니다. 이후 시간이 난다면 다시 리팩토링하여 이전 코드와 이후 코드를 비교하여 확인해보려합니다.

  • 해결해야할 기능 구조
    1. 드래그 이벤트를 작성해야합니다. 지금은 클릭 이벤트로 바꾸었으나 이후 mouseup, mousedown을 사용하여 마우스의 screenY의 값을 추적하여 사용하고 싶습니다. 다만 이 기능의 경우 우선순위가 떨어지므로 이후 다른 페이지를 먼저 작업하고 후 순위로 작업하려 합니다.
    2. 코드의 리팩토링을 원하나 남은 기간이 적고 제출물이 남은 상황이므로 드래그 이벤트까지 작성 후 리팩토링을 진행할 예정입니다.
    3. 이후 chart.js를 사용하여 차트 페이지를 제작하고 이체하는 화면도 제작해야합니다.
