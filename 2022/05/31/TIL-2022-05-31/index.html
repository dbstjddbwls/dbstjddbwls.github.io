수업 내용

Babel

  • Javascript Compoiler 중 하나로 현재 사용중인 언어를 low한 언어로 바꿔주는 일반적인 compiler와 달리 babel은 transpiler로 es6의 문법을 es5도 사용 가능하게 번역을 해주는 역할을 한다.
  • 최신 브라우저를 사용 중이지 않은 사용자에게도 동일한 경험을 제공할 수 있다.
  • npm install @babel/core @babel/cli @babel/plugin-transform-arrow-functions 등 명령어를 통하여 설치하여 사용한다.
  • .babelrc란 파일을 통하여 보통 관리한다.

회원가입 실습

  • axios를 이용한 코드 작성 실습
  • 조건
    • ID 중복체크 기능
    • 회원가입 기능
  • 추가 작업 사항
    • 로그인 되지 않은 상태로 회원 이용 페이지 접속 시 login 페이지로 리다이렉트
    • 로그인 된 상태로 로그인 창이나 회원가입 창 이동 시 main 페이지로 리다이렉트
